>>> composer install

>>> cp .env.example .env
>
>> 3. Generamos una key nueva
>>> php artisan key:generate
>
>> 4. Corremos las migraciones
>>> php artisan migrate

>>> php artisan db:seed

>> 7. Actualizar las tablas
